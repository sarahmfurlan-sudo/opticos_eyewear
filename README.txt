OPTICOS EYEWEAR
Sistema de Gestão de Estoque
FUNCIONALIDADES DO SISTEMA

O sistema permite:

- Cadastrar novos produtos
- Listar todos os produtos cadastrados
- Atualizar a quantidade em estoque
- Remover produtos do sistema
- Buscar produtos pelo código
- Verificar produtos com estoque baixo
- Gerar relatório completo do estoque
- Salvar os dados em arquivo
- Carregar automaticamente os produtos ao iniciar o sistema

INFORMAÇÕES DE CADA PRODUTO

Cada produto cadastrado possui os seguintes dados:

- Nome do produto
- Código do produto
- Categoria
- Quantidade em estoque
- Preço
- Fornecedor


ESTRUTURA DO PROJETO

main.py  
Arquivo principal responsável pela execução do sistema e exibição do menu.

produto.py  
Arquivo que contém a definição da classe Produto.

funcoes_estoque.py  
Arquivo responsável pelas funções que executam as operações do sistema.

produtos.txt  
Arquivo utilizado para armazenar os dados dos produtos cadastrados.
